# 🎄 奢华交互 3D 圣诞树 | Grand Luxury Interactive 3D Tree

> 一个基于 **React**, **Three.js (R3F)** 和 **AI 手势识别** 的高保真 3D 记忆画廊。

本项目是一个融合了科技与仪式感的圣诞主题 Web 应用。用户可以上传自己的照片，通过先进的 AI 手势识别技术，在 3D 空间中与承载记忆的照片进行互动，体验电影级的视觉盛宴。

![Project Preview](public/preview.png)
*(注：项目采用了黑色极简高级风格，配以金色的星辰与祖母绿的松针粒子)*

---

## ✨ 核心特性

*   **🎬 电影级视觉体验**：由 15,000+ 个发光粒子组成的树身，配合动态光晕 (Bloom)、HDR 夜景环境贴图和 3D 线条装饰，营造出极致的奢华氛围。
*   **📸 专属记忆画廊**：
    *   **动态上传**：支持前端直接上传照片（最多 31 张），照片以“拍立得”风格悬浮在树身周围。
    *   **交互查看**：点击圣诞树上的任意照片，即可弹出丝滑的沉浸式大图预览。
    *   **循环逻辑**：如果照片少于 31 张，系统会自动循环填充所有挂件位，确保美观。
    *   **无照片模式**：支持一键开启纯粒子/线条风格，感受纯粹的 3D 艺术美感。
*   **🖐️ 全方位 AI 手势交互**：内置 MediaPipe AI 视觉引擎，无需触摸屏幕即可掌控全场：
    *   **形态控制**：🖐️ **张开手掌**（散开粒子/照片） | ✊ **紧握拳头**（聚拢成树）。
    *   **视角旋转**：👋 **手掌左右移动**，带动圣诞树 360 度平滑旋转。
    *   **俯仰控制**：👋 **手掌上下移动**，实现视角的抬升与降低。
    *   **智能缩放**：📏 **手掌前后移动**，圣诞树随之放大或缩小。
*   **🖱️ 智能双模控制**：系统自动识别手部状态。当手放下时，自动恢复鼠标控制权（支持鼠标拖拽旋转、滚轮缩放、右键平移）。
*   **⚡ 极致性能优化**：前端实时处理/压缩超大尺寸图片，显著降低显存占用，防止 GPU 崩溃黑屏。

*   **框架**: React 18, Vite

*   **AI 视觉**: MediaPipe Tasks Vision (Google)

*   **后期处理**: @react-three/postprocessing (Bloom, Vignette)

## 🚀 快速开始

### 1. 环境准备
确保你的电脑已安装 [Node.js](https://nodejs.org/) (建议 v18 或更高版本)。

### 2. 安装并运行
在项目根目录下打开终端：

```bash
# 安装依赖
npm install

# 启动本地开发服务器
npm run dev
```

### 3. 部署上线
本项目支持一键部署至 Vercel、Netlify 或 GitHub Pages。建议关联 GitHub 仓库后在 Vercel 导入，所有配置均已预设。

## 🖐️ 手势操作指南

请确保你站在摄像头前，并在场景中点击“📷 摄像头画面”以确认识别状态：

| 手势 | 动作 | 功能说明 |
| :--- | :--- | :--- |
| **张开手掌** | 🖐️ | **散开**：圣诞树炸裂成漫天飞舞的粒子和照片 |
| **紧握拳头** | ✊ | **聚合**：所有元素瞬间聚合成完美的圣诞树 |
| **左右平移** | 👋 | **旋转**：手向左，树左转；手向右，树右转 |
| **上下平移** | 👋 | **俯仰**：手向上，视角降低；手向下，视角抬高 |
| **前后推拉** | 📏 | **缩放**：手靠近镜头放大，远离镜头缩小 |
